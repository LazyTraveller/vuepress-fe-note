### 运行项目
```sh
 # 安装依赖
 1、yarn 

 # 启动项目
 2、yarn docs:dev 
 
 # 内容贡献
 3、在本地添加或编辑 demo.md 文件
 
 # 发布
 4、yarn deploy-gh 一步到位发布版本， 在 github page 即可访问最新的内容

 5、把代码推到 master 即可

```
