### 一个前端知识笔记


##### 宗旨：在记录过程中不断成长进步, 一分耕耘，一分收获

##### 原则：简单化，一切问题都可以简单化

##### 内容贡献流程


```
 1、yarn 

 2、yarn docs:dev 在本地添加或编辑 demo.md 文件
 
 3、yarn deploy-gh 一步到位发布版本， 在 github page 即可访问最新的内容

 4、把代码推到 master 即可

```
